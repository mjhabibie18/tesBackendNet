## Tips Menghadapi Technical Test

1. **Baca soal dengan teliti** — jangan terburu-buru
2. **Tanya klarifikasi** — edge cases, format output
3. **Think out loud** — jelaskan pemikiran Anda
4. **Mulai dari brute force** — optimasi belakangan
5. **Test dengan contoh** — manual trace sebelum submit
6. **Manage waktu** — jangan stuck di satu soal

---

## Tips Menghadapi Interview

1. **STAR Method**: Situation, Task, Action, Result
2. **Jujur tentang keterbatasan**: lebih baik dari pura-pura tahu
3. **Tanya balik**: tunjukkan keingintahuan
4. **Siapkan proyek portofolio**: lebih baik dari teori saja
5. **Latihan di LeetCode** minimal Easy/Medium
